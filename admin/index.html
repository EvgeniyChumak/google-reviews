Google Reviews Script Generator

You can find the Place ID of any business here: Google Place ID Finder

Final 2 Steps

  1. Copy the generated code and paste it into:
    Project Settings → Custom Code → Before </body> (if you need the rating on all pages)

    or

    Page Settings → Custom Code → Before </body> (if you need the rating only on one page)
  2. Add attributes [google-rating] and [google-total] to the elements that should display the rating and total reviews.

    Tip: wrap only the needed text in a <span> and place the attribute on it.
